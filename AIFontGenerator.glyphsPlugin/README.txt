AI Font Generator for Glyphs.app
=================================

Generate a complete font from sample glyphs using AI.

INSTALLATION
------------
1. Double-click AIFontGenerator.glyphsPlugin to install
   Or copy to: ~/Library/Application Support/Glyphs 3/Plugins/

2. Restart Glyphs.app

USAGE
-----
1. Open a .glyphs file with at least one designed glyph
2. Select the glyph(s) to use as style reference
3. Go to Filter menu > AI Generate Full Font
4. Choose your style reference source and output mode
5. Click Generate
6. Wait a few minutes for AI generation

REQUIREMENTS
------------
- Glyphs.app 3.x
- macOS 10.14 or later
- Internet connection
